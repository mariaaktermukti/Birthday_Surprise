# 🎉 Birthday Surprise

A simple and interactive birthday surprise web application built using HTML, CSS, and JavaScript.

## ✨ Features
- Birthday greeting
- Interactive animations
- Beautiful UI
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/mariaaktermukti/Birthday_Surprise.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
